| ┌──────────────────────┬───────────────────────────────────────────────────────────────┐ |
| │      COMPONENT       │                  DESCRIPTION & USE (Istemal)                  │ |
| ├──────────────────────┼───────────────────────────────────────────────────────────────┤ |
| │ 🧩 Motherboard       │ Main Backbone. Yeh ek bara green/black board hota hai jis se  │ |
| │                      │ computer ke saare internal parts aapas me connect hote hain.  │ |
| │ 🧠 CPU (Processor)   │ Brain of Computer. Saari calculations aur processing isi me    │ |
| │                      │ hoti hai. (e.g., Intel Core i5, AMD Ryzen).                   │ |
| │ 💾 RAM (Memory)      │ Short-term Memory. Jab tak computer ON hai, yeh chalne wale   │ |
| │                      │ apps ka data temporary hold karti hai. Fast but volatile.     │ |
| │ 🗄️ Storage           │ Long-term Memory. Aapka permanent data (OS, Files, Videos)    │ |
| │    (SSD / HDD)       │ yahan save hota hai. SSDs purane HDDs se bohot tez hoti hain. │ |
| │ 🎮 GPU (Graphics)    │ Video/Image Processor. Screen par display dikhane, gaming,     │ |
| │                      │ aur heavy video editing ko handle karta hai.                  │ |
| │ 🔌 PSU (Power        │ Power Supply Unit. Board aur baki parts ko sahi voltage ki    │ |
| │    Supply)           │ electricity/bijli supply karta hai.                           │ |
| └──────────────────────┴───────────────────────────────────────────────────────────────┘ |
