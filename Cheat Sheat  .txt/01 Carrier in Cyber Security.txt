|                       CYBER SECURITY CAREER PATHWAYS & FIELDS                            |
|                      =========================================                           |
|                                                                                          |
|   Cyber Security ka maqsad networks, devices, programs, aur data ko cyber attacks,     |
|   damage, ya unauthorized access se safe rakhna hai. Isme career ke bohot se raste hain.|
|                                                                                          |
| ┌──────────────────────────────────────────────────────────────────────────────────────┐ |
| │                      1. THE TWO MAIN SIDES (Main Teamon Ke Raste)                    │ |
| ├──────────────────────────────────────────────────────────────────────────────────────┤ |
| │  🔴 RED TEAM (Offensive Security / Ethical Hacking)                                  │ |
| │  - Inka kaam hackers ki tarah sochna aur system me khamiyan (vulnerabilities)        │ |
| │    dhoondna hota hai taake unhe pehle hi theek kiya ja sake.                         │ |
| │  - Job Roles: Ethical Hacker, Penetration Tester, Vulnerability Assessor.            │ |
| │                                                                                      │ |
| │  🔵 BLUE TEAM (Defensive Security / System Protection)                               │ |
| │  - Inka kaam system ko 24/7 monitor karna, attacks ko rokna, aur agar attack ho      │ |
| │    jaye to us par foran action lena hota hai.                                         │ |
| │  - Job Roles: SOC Analyst, Security Engineer, Incident Responder.                    │ |
| └──────────────────────────────────────────────────────────────────────────────────────┘ |
|                                                                                          |
| ┌──────────────────────────────────────────────────────────────────────────────────────┐ |
| │                      2. CORE CYBER SECURITY JOB ROLES                                │ |
| ├──────────────────────────────────────────────────────────────────────────────────────┤ |
| │  * SOC Analyst (Tier 1/2/3): Network traffic ko monitor karna aur ajeeb activity ko  │ |
| │    pakadna. Entry-level ke liye behtareen job hai.                                   │ |
| │  * Penetration Tester: Company ki ijazat se unke networks par "fake attack" karna    │ |
| │    taake security check ki ja sake.                                                  │ |
| │  * Digital Forensics Expert: Cyber crime hone ke baad investigation karna, data       │ |
| │    recover karna, aur saboot (evidence) dhoondna.                                    │ |
| │  * CISO (Chief Information Security Officer): Kisi bhi company me cyber security ka  │ |
| │    sab se bada boss jo poori security strategy banata hai (Top Executive level).     │ |
| └──────────────────────────────────────────────────────────────────────────────────────┘ |
