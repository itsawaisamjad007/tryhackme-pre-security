|                       OPERATING SYSTEM (OS): BASIC LAYOUT & FUNCTIONS                    |
|                      =================================================                   |
|                                                                                          |
|   Operating System (OS) ek master program hai jo computer hardware ko manage karta hai   |
|   aur user software (Apps) ko chalane ke liye ek platform mohayya karta hai.             |
|                                                                                          |
| ┌──────────────────────────────────────────────────────────────────────────────────────┐ |
| │                            OPERATING SYSTEM ARCHITECTURE                             │ |
| ├──────────────────────────────────────────────────────────────────────────────────────┤ |
| │         [ USER ] ◄───► [ APPLICATIONS ] ◄───► [ OS ] ◄───► [ HARDWARE ]              │ |
| │     (Hum log)         (Chrome, Games)     (Win/Linux)    (CPU, RAM, Disk)            │ |
| └──────────────────────────────────────────────────────────────────────────────────────┘ |
|                                                                                          |
| ┌──────────────────────┬───────────────────────────────────────────────────────────────┐ |
| │    MAIN FUNCTION     │                  DESCRIPTION & USE (Istemal)                  │ |
| ├──────────────────────┼───────────────────────────────────────────────────────────────┤ |
| │ 🧠 Process           │ CPU scheduling karna. Pata lagana ke kaun sa program kab aur  │ |
| │   Management         │ kitni der ke liye CPU istemal karega.                         │ |
| │ 💾 Memory            │ RAM ko manage karna. Jab koi app khulti hai to use jagah dena │ |
| │   Management         │ aur band hone par RAM khali karwana.                          │ |
| │ 🗂️ File              │ Data ko storage (SSD/HDD) me sahi tarike se files aur folders │ |
| │   Management         │ ki shakal me save aur delete karne ka kam.                    │ |
| │ 🔌 Device            │ Hardware devices (Mouse, Keyboard, Printer) ke drivers ko     │ |
| │   Management         │ control karna taake wo computer ke sath sahi kaam karein.     │ |
| │ 🛡️ Security          │ Passwords aur firewall ke zariye unauthorized users aur       │ |
| │                      │ malware/viruses se system ko mahfooz rakhna.                  │ |
