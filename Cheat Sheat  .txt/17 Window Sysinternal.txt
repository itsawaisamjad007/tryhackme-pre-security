|                                                                                          |
|                     WINDOWS SYSINTERNALS SUITE: ADVANCED UTILITIES                       |
|                    ================================================                      |
|                                                                                          |
|   Sysinternals aese advanced tools hain jo Windows ke normal functions (jaise Task       |
|   Manager) se hazar guna zyada gehrai (under-the-hood) me ja kar data dikhate hain.      |
|                                                                                          |
| ┌──────────────────────────────────────────────────────────────────────────────────────┐ |
| │                       THE "BIG THREE" MUST-KNOW UTILITIES                            │ |
| ├──────────────────────────────────────────────────────────────────────────────────────┤ |
| │  1. Process Explorer (procexp.exe)  -- [Super Task Manager]                           │ |
| │     - Ye dikhata hai ke kaun sa process kis file (DLL) ko use kar raha hai aur kis   │ |
| │       parent process ne use launch kiya. Malware detection ke liye sab se zaroori hai.│ |
| │                                                                                      │ |
| │  2. Process Monitor (procmon.exe)  -- [Real-time System Activity Tracker]             │ |
| │     - Real-time me registry changes, file system access, aur process execution ko     │ |
| │       capture karta hai. Isme filter lagana sab se ahem skill hai.                   │ |
| │                                                                                      │ |
| │  3. Autoruns (autoruns.exe)  -- [Startup Inspector]                                   │ |
| │     - Computer start hote hi chalne wale har ek program, service, browser extension, │ |
| │       aur driver ki list dikhata hai. Hackers aksar yahan apna malware chupate hain. │ |
| └──────────────────────────────────────────────────────────────────────────────────────┘ |
|                                                                                          |
| ┌──────────────────────┬───────────────────────────────────────────────────────────────┐ |
| │     OTHER TOOLS      │                  DESCRIPTION & USE (Istemal)                  │ |
| ├──────────────────────┼───────────────────────────────────────────────────────────────┤ |
| │ 📁 AccessChk         │ Check karta hai ke kisi specific user ko kis file, folder, ya │ |
| │                      │ registry key par kya permissions (read/write) mili hui hain.  │ |
| │ 🔗 TCPView           │ Computer par hone wali har active network connection ko live   │ |
| │                      │ dikhata hai ke kaun sa app kis IP address se baat kar raha hai│ |
| │ 🛡️ Sysmon            │ System Monitor. Background service jo system events ko deep    │ |
| │   (System Monitor)   │ level par event log me record karti hai (for security SOC).   │ |
| │ 🔍 Strings           │ Kisi bhi executable (.exe) file ke andar chupe hue plain text │ |
| │                      │ (IPs, domains, messages) ko extract karne ke liye command tool│ |
