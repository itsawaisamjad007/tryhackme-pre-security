 Virtualization Basic :-
                       To make 1 computer into 10 computers . We can
                       run Windows + Linux + Mac at a same time .If
                       VM crash the real PC will not affected.

       For Example :
                    If we have PC With RAM 16 GB and 1TB SSD and 8
                    Core CPU . we can create 4 VM as :
                       * VM1 : 4 GB RAM , 2 Core
                       * VM2 : 4 GB RAM , 2 Core
                       * VM3 : 4 GB RAM , 2 Core
                       * VM4 : 4 GB RAM , 2 Core

  Benefits :-
             * We can copy any any VM in pen drive and and can open
             into any other laptop . For this purpose we copy the
             files of VM from location where it is installed . And
             then paste on any other laptop open VM Box into those
              Laptop and open here.

             * We can take snapshot op VM . When VM is 100 % correct
             We can take snapshot even VM is on or off , But off
             way is best . First we off the VM . Then in virtual box
             double click on VM and click snapshot and then further
             processing . If anytime VM is corrupted click restore
             option on VM and it will be take backup to first correct
             snapshot . (Do not create multiple snapshot because it
             take storage in SSD and slow the PC) .
