
  Server :-
           It is a powerful computer which give service and
           store data .
             * Web server
             * Database server
             * File Server
             * Google server
             * Facebook server
             * youtube server
     Its working is to receive request from client search
     data and send to the client

  Client :-
            It is a device which request data from server.
                 * Mobile
                 * Laptop
                 * Chrome Browser

  Request-Response Cycle :-
                          When we search any thing on search
                          engine or browser it become a HTTP
                          request and when server reply us it
                          receive in the form of HTTP response .
