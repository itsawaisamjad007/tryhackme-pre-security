    Cloud Service Model :-
                          * Iaas :
                                 Infrastructure as a Service . In this
                                 service model cloud provide you RAM , Disk ,Network
                                 and Electricity . And you use your OS , Apps,
                                 Data , update and security. in it user have max
                                 control. If we need Kali VM for pentesting we
                                 have to use IaaS because we have to full
                                 control of OS.
                          * PaaS :
                                  Platform as a Service . In this service
                                  model Cloud provide OS , Database , Runtime
                                  , Server and Network . you use only code
                                    and data. In it user have middle control.
                                    If I have to create Python website I use
                                    PaaS because we have to only use my code
                                    and data nothing more and Cloud will
                                    manage left all.
                          * Saas :
                                   Software as a Service . In this service
                                   model cloud provide you everything you
                                   just have to login and use it . Like
                                   google drive , Gmail , Netflix ,ChatGPT
                                   In it user have very less control . If
                                   I need Email I use SaaS because I have
                                   not to control of server.

    Cloud Deployment Models :-
                          * Public Cloud :
                                          Which can use any company in
                                          the world , It is cheaper , It
                                          has less security control . Like
                                          Startup , Website , Netflix

                          * Private Cloud :
                                           For 1 company only . It provide
                                           full control , good security level
                                           . It is very expensive , Like in
                                           Bank , Hospital , Government , Army.

                          * Hybrid Cloud :
                                          Some public and some private . Like
                                          Company or Bank costumer data in private
                                          and website in public . It is difficult
                                          to manage . It may be costly or cheaper.

                          * Community Cloud :
                                             For 2-3 companies in combine ,
                                             In this model cost is share among
                                             them , But is available for specific
                                             peoples . For Example all hospitals
                                             or all universities have 1 cloud .

    Advantages :-
                * On Demand :
                             We can get 50 server on one click .
                * Scalability :
                               If traffic increase server will also
                               auto increase .
                * Pay as you go :
                                 We will only pay for which we use.
                * Reliability :
                               If one data centre damage then other
                               run .
                * Global :
                          create VM om one click in any country.

    Disadvantages :-

                            Data is run on company server not on

                * Internet need :
