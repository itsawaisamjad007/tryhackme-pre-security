
When we press power button :-
                            * Press power button :-
                                                When we press power
                             button a signal is set to PSU to allow
                             power to flow.

                            * Firmware start :-
                                              The software which run after
                              power button in system , It is a software
                              which is permanently stored in the hardware
                              in the motherboard chip . BIOS (Basis input
                              and output system)  is old firmware since 1980
                              UEFI (Unified Extensible Firmware Interface)
                              is a new Firmware since 2010 , and now it
                              is basically installed in every PC and
                              Laptop

                            * POST :-
                                     POST (Power on self test) It is first
                             step of firmware to test CPU , GPU , Keyboard
                             , mouse , Hard disk , Motherboard , why all
                             is well. If all is well then PC goes to boot.

                            * Select boot device :-
                                                  In this step PC ask where
                             where your operating system or window is placed
                             . This interface is open after POST by using
                             F12 , F9 , Fn + F9 ,F8 or F11 according brand of
                             PC

                            * Start boot Loader :-
                                                     This bootloader transfers
                             the Operating System from the selected boot device
                             to the Random Access Memory. Once the OS is transferred
                             , the UEFI gives control over the different components
                              to the OS. In window boot loader is named (windows boot
                              manager), In LINUX it is named as GRUB (Grand Unified
                              Bootloader)

 1 .Press Power Button :-
                        You press the power button, sending a signal to the PSU to
                         let power through. The system begins powering up.

2
Firmware Starts (UEFI/BIOS) :-
                           UEFI (or older BIOS) firmware initializes and coordinates
                            components. It's the first software that runs.
3
Power-On Self Test (POST) :-
                          UEFI runs POST to check that required hardware is present,
                           configured, and functioning. Errors trigger beeps or alerts.

4
Select Boot Device :-
                        UEFI follows a priority list to determine which device to boot
                         from - typically an SSD or HDD with the OS installed.
5
Initiate Bootloader :-
                       The bootloader loads the operating system into RAM. Then UEFI
                        control to the OS, completing the boot sequence.
